# CMake Playground

This is a simple C++ project to test the capabilities of CMake & CPack in combination with various generators. It's aim is to provide a template for a fully automated C++ build environment that:

- builds on various platforms
- generates installers for various platforms
- has a fully automated release process
- runs static code analysis
- validates code style

The current project setup contains a simple C++ string manipulation library. Furthermore there is a test project and an executable that uses the library. The C++ source code is mostly generated by AI because it is not the main focus of this project. The main focus is the project setup and structure.

## Disclaimer

This is not complete and may have errors and hiccups. It is based on my knowledge and experience with these tools over the last few years. This is probably not perfect and may not always follow best practices, but I tried to stick to them as good as possible.

Mainly this project serves as a notepad for me, which I thought might be useful for others as well. So feel free to leave comments and improvement requests.
